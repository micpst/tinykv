## 🛠️ Installation

### Build from source
To build and run fts-engine from the source code:
1. Requirements: **go & make**
2. Install dependencies:
```bash
$ make setup
```
3. Build:
```bash
$ make build
```
4. Run the server binary:
```bash
$ ./bin/server
```
